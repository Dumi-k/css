# Типы селекторов CSS

### Универсальный селектор
С помощью данного селектора можно применить стили ко всем элементам веб-страницы. Например:

`css`
```css
* {color: red}
```
    _Это правило делает содержимое всех элементов страницы красным цветом._
***

### Селектор по тегу

**html**
```html
<p>Далеко-далеко за словесными горами в стране.</p>
```
**css**
```css
p {color: red}
```
